# index-hbzb-resource

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态资源的形式存储，不针对任何特定域名或网站，主要作为通用 HTML 文档的集合与存档。

## 目录说明

```
index-hbzb-resource/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面依赖的样式、脚本、图片等资源
├── docs/           # 相关文档或说明
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件对应一个独立页面，文件命名清晰，便于查找。
- `assets/`：按需存放页面引用的外部资源，保持目录结构整洁。
- `docs/`：可选，用于存放额外的使用说明或技术笔记。

## 页面归档说明

- 所有页面均为纯静态 HTML，支持直接通过浏览器打开或部署到任意静态托管服务。
- 页面之间相互独立，无交叉依赖，便于单独引用或分发。
- 归档内容不包含任何动态后端逻辑，仅保留前端展示与交互。

## 维护说明

- 欢迎提交 Pull Request 增加新的 HTML 页面或优化现有页面。
- 提交前请确保页面无外部敏感链接，并保持文件结构清晰。
- 本仓库不承诺定期更新，但会尽力维护归档的完整性与可访问性。
